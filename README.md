# Physics2DPlayground

Simple 2D grid physics simulator. There are two implementations - C++/Qt-based and Qml-based.

Controls:
- vertex drag & drop - Left Mouse Button
- space moving - Middle Mouse Button
- zoom in / zoom out - Mouse Wheel
